## Общая Концепция Дизайна

### Ключевые Визуальные Принципы
- **Циркуитная эстетика**: Нейронные соединения, потоки данных, электрические импульсы
- **Пошаговая прогрессия**: Каждый клик показывает следующий этап генерации
- **Математическая точность**: Схемы, графики, численные данные
- **Контраст скоростей**: Миллисекунды vs сложность процессов

## **СЛАЙД 12.1** — Автогенеративный Цикл

### Общая Композиция
**Формат**: Горизонтальная схема цикла с прогрессивным раскрытием
**Время на слайде**: 90 секунд
**Количество кликов**: 3

### Начальное Состояние
```
[Заголовок]
ГЕНЕРАЦИЯ: СЛОВО ЗА СЛОВОМ
Как ИИ выбирает каждое следующее слово

[Центральная область - пуста, готова к раскрытию]
```

### **КЛИК 1**: Исходный Контекст
```
┌─────────────────────────────────┐
│  "Искусственный интеллект может" │  ─→  [?]
└─────────────────────────────────┘
     ↑
  Входной контекст
```

**Визуальные элементы**:
- Текстовый блок с закругленными углами
- Стрелка, показывающая направление обработки
- Знак вопроса в блоке предсказания
- **Цвет**: `--generation-primary` для основных элементов

### **КЛИК 2**: Вероятностная Обработка
```
┌─────────────────────────────────┐       ┌──────────────────┐
│  "Искусственный интеллект может" │  ─→   │  50,000 токенов  │
└─────────────────────────────────┘       │                  │
     ↑                                    │  "помочь" - 23%   │
  Входной контекст                        │  "решать" - 15%   │
                                          │  "создать" - 12%  │
                                          │  "понять" - 8%    │
                                          │  ... остальные    │
                                          └──────────────────┘
```

**Визуальные элементы**:
- Расширенный блок с примерами вероятностей
- Полупрозрачные полосы разной длины для визуализации вероятностей
- **Цвета**: высокие вероятности `--probability-high`, низкие `--probability-low`

### **КЛИК 3**: Выбор и Новый Цикл
```
┌─────────────────────────────────┐       ┌──────────────────┐       ┌────────────────────────────────────┐
│  "Искусственный интеллект может" │  ─→   │  Выбор: "помочь"  │  ─→   │  "Искусственный интеллект может помочь" │
└─────────────────────────────────┘       └──────────────────┘       └────────────────────────────────────┘
     ↑                                           ↓                                    ↓
  Исходный контекст                    Необратимый выбор              Новый контекст для следующего слова
```

**Визуальные элементы**:
- Зеленая подсветка выбранного варианта (`--circuit-active`)
- Стрелка цикла, показывающая возврат к началу процесса
- Анимация "пульсирования" для показа активности

### Ключевые Аннотации
- **"НЕОБРАТИМО"** — красная подпись под выбранным токеном
- **"~10 мс"** — временная метка процесса
- **"50K вариантов"** — масштаб вычислений

## **СЛАЙД 12.2** — Температура и Креативность

### Общая Композиция
**Формат**: Интерактивная шкала с примерами
**Время на слайде**: 90 секунд
**Количество кликов**: 3

### Начальное Состояние
```
[Заголовок]
ТЕМПЕРАТУРА: КОНТРОЛЬ КРЕАТИВНОСТИ
От предсказуемости к неожиданности

[Вертикальная шкала температуры - пуста]
```

### **КЛИК 1**: Низкая Температура (0.1)
```
Температура: 0.1  ┃█████████████████████  "помочь" (85%)
ПРЕДСКАЗУЕМОСТЬ   ┃██████                 "решать" (10%)
                  ┃██                     "создать" (3%)
                  ┃█                      "понять" (2%)
                  ┃▌                      другие...

Пример: "может помочь в решении сложных задач"
```

**Визуальные элементы**:
- Длинные полосы для популярных вариантов
- Короткие для остальных
- Пример текста в рамке снизу
- **Цвет**: `--probability-high` для доминирующих вариантов

### **КЛИК 2**: Нормальная Температура (1.0)
```
Температура: 1.0  ┃██████████████         "помочь" (35%)
СБАЛАНСИРОВАННОСТЬ┃████████████           "решать" (25%)
                  ┃████████               "создать" (18%)
                  ┃█████                  "понять" (12%)
                  ┃███                    другие (10%)

Пример: "может создавать инновационные решения"
```

**Визуальные элементы**:
- Более равномерное распределение полос
- Изменение примера текста
- Плавная анимация перехода между состояниями

### **КЛИК 3**: Высокая Температура (1.5)
```
Температура: 1.5  ┃███████                "помочь" (20%)
КРЕАТИВНОСТЬ      ┃███████                "решать" (18%)
                  ┃███████                "создать" (17%)
                  ┃██████                 "понять" (15%)
                  ┃██████████████         другие (30%)

Пример: "может танцевать между возможностями реальности"
```

**Визуальные элементы**:
- Почти равномерные полосы
- Заметная доля для "других" вариантов
- Более творческий пример текста
- **Цвет**: радужный градиент для показа разнообразия

### Боковая Аннотация
```
┌─────────────────────────────┐
│ Математически:              │
│ P(токен) = exp(логит/T)     │
│                             │
│ T ↓ → концентрация          │
│ T ↑ → разнообразие          │
└─────────────────────────────┘
```

## **СЛАЙД 12.3** — Пошаговая Анатомия Генерации

### Общая Композиция
**Формат**: Временная диаграмма с детализацией процессов
**Время на слайде**: 60 секунд
**Количество кликов**: 4

### Начальное Состояние
```
[Заголовок]
АНАТОМИЯ 10 МИЛЛИСЕКУНД
Что происходит при генерации одного слова

[Временная шкала 0-10 мс, пуста]
```

### **КЛИК 1**: Кодирование (0-1 мс)
```
0ms ├──────┤ 1ms
    │КОДИРОВАНИЕ│

┌─────────────────────────────────┐
│ • Токены → векторы              │
│ • Позиционные кодировки         │
│ • Подготовка к обработке        │
└─────────────────────────────────┘
```

**Визуальные элементы**:
- Синяя заливка сегмента времени
- Схематичное изображение векторов
- Простые пиктограммы процессов

### **КЛИК 2**: Слоевая Обработка (1-6 мс)
```
0ms ├──────┤ 1ms ├────────────────┤ 6ms
    │КОДИРОВАНИЕ│   СЛОИ 1-12         │

┌─────────────────────────────────┐
│ 12 слоев × Self-Attention       │
│                                 │
│ Слой 1  ┃█░░░░░░░░░░░░  Синтаксис│
│ Слой 6  ┃██████░░░░░░  Семантика│
│ Слой 12 ┃████████████ Контекст │
└─────────────────────────────────┘
```

**Визуальные элементы**:
- Зеленая заливка для основной обработки
- Прогресс-бары для каждого типа слоев
- Нейронная сеть в миниатюре

### **КЛИК 3**: Генерация Логитов (6-9 мс)
```
0ms ├──────┤ 1ms ├────────────────┤ 6ms ├───────┤ 9ms
    │КОДИРОВАНИЕ│   СЛОИ 1-12         │ ЛОГИТЫ  │

┌─────────────────────────────────┐
│ Внутренне представление → 50K числа │
│                                 │
│ "помочь": 2.1                   │
│ "решать": 1.8                   │
│ "создать": 1.5                  │
│ ...                             │
└─────────────────────────────────┘
```

**Визуальные элементы**:
- Фиолетовая заливка
- Схема превращения векторов в числа
- Примеры логитов с числовыми значениями

### **КЛИК 4**: Выборка Токена (9-10 мс)
```
0ms ├──────┤ 1ms ├────────────────┤ 6ms ├───────┤ 9ms ├─┤ 10ms
    │КОДИРОВАНИЕ│   СЛОИ 1-12         │ ЛОГИТЫ  │С│

┌─────────────────────────────────┐
│ Применение температуры          │
│ Softmax → вероятности           │
│ Случайная выборка               │
│                                 │
│ Результат: "помочь" ✓           │
└─────────────────────────────────┘
```

**Визуальные элементы**:
- Красная заливка финального этапа
- Символ случайности (кости/колесо рулетки)
- Галочка у выбранного токена
- **Анимация**: "выстрел" выбранного токена

### Итоговая Статистика
```
┌─────────────────────────────────┐
│ ИТОГО ЗА 10 МИЛЛИСЕКУНД:        │
│ • ~1 триллион операций          │
│ • 50,000 вероятностей           │
│ • 1 выбранное слово             │
│                                 │
│ Скорость: 10-100 слов/секунду   │
└─────────────────────────────────┘
```

## Технические Спецификации

### Анимации и Переходы
- **Появление элементов**: Плавное fade-in за 0.3 секунды
- **Пульсирование активности**: 1.5 секунды цикл
- **Прогресс-бары**: Заполнение за 1 секунду
- **Переходы между кликами**: 0.5 секунды

### Типографика
- **Заголовки слайдов**: 28pt, полужирный, `--text-primary`
- **Основной текст**: 18pt, обычный
- **Аннотации**: 14pt, курсив
- **Числовые данные**: 16pt, моноширинный шрифт

### Адаптивность
- **Для больших экранов**: Увеличенные схемы, больше деталей
- **Для средних залов**: Стандартный размер
- **Для удаленки**: Упрощенные схемы, крупнее текст

### Интерактивные Элементы
- **Hover-эффекты**: Подсветка элементов при наведении
- **Кликабельные зоны**: Ясно обозначенные области для продвижения
- **Прогресс-индикатор**: Показ текущего положения в слайде
